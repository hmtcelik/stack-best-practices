# Stack Best Practices in C

### Stack data structure best practices in C with these forms:

- Linked List
- Array

#### There are 3 function to manipulation of stack:

```C
void init(Stack *stk);
/*
  init(stack) -> void 
*/

void push(Stack *stk, int value);
/*
  push(stack, 2) -> void 
*/

int pop(Stack *stk);
/*
  pop(stack) -> 2 (top element) 
*/
```
